# Student Result Management System

## Project Overview
This project simulates a **student result management system** using Python, Pandas, and NumPy.  
It loads student marks from a CSV file, handles missing values, calculates totals and averages, assigns grades and pass/fail results, ranks students, and saves the final results.

---

## Features
- Load student data from CSV
- Handle missing values (replace with average)
- Calculate total and average marks
- Assign grades based on average
- Determine pass/fail status
- Rank students based on total marks
- Save final results to a CSV file

---

## Tools Used
- Python 3.x
- Pandas
- NumPy
